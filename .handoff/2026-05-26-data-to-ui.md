# UI Agent: 新数据字段可用于卡片增强

**From**: Data Agent
**To**: UI Agent
**Date**: 2026-05-26

## 新可用数据字段

### interviewHistory — 面试记录

resumes.json 中 7 位候选人有结构化面试记录（date + rating 1-5 + feedback 全文）

**卡片影响**: C2 ProfileCard 可展示面试时间线 + 评分星级 + 反馈摘要；C3 ComparisonCard 可加入"面试表现"对比维度

### referrals.json — 内推关系（24 条）

含推荐人姓名、推荐语全文、关系类型、状态

**卡片影响**: C2 中展示"内推来源"行；可选新建内推网络小组件

### notes 深化

所有黄金候选人 notes 补充了：内推来源、离职动机、薪酬博弈信息、竞品竞争

### online_presence 丰富

GitHub/blog/linkedin 覆盖更全，可用于 C2 在线足迹区块

## 数据位置

| 数据 | 文件 | 字段路径 |
|------|------|----------|
| 面试记录 | resumes.json | data[].interviewHistory |
| 内推 | referrals.json | referrals[] |
| 备注 | resumes.json | data[].notes |
| 在线足迹 | resumes.json | data[].onlinePresence |
