## 设计决策

### contracts 对齐
- 严格以 `contracts/cards.ts` 的 `CardBase` 接口为准
- `mode: CardMode`（loading/empty/error/demo/live）作为状态字段
- `CardAction` 用 `label / message / variant`，点击回调 `onActionClick(message)`
- `is_demo` 独立于 mode，允许 `mode='live' + is_demo=true` 叠加标识

### C9 记忆唤醒卡
- 用 framer-motion `animate boxShadow` 实现呼吸金边效果（DSP-3 视觉爆点）
- `MemoryRecallItem.layer` 用图标区分：session=MessageCircle(蓝)、user=User(紫)、candidate=UserCheck(绿)

### Demo 标识
- `mode='demo'` 时强制渲染演示数据 + DemoBadge
- `mode='live'` + `is_demo=true` 时仅叠加 DemoBadge（无演示数据，适用于真实数据流测试）
